# MyApp013aEducationGame (Cyber Defender)
Edukační Android hra s tématikou kybernetické bezpečnosti. Hráč vystupuje jako agent, který brání systém před viry správným zodpovídáním otázek. Aplikace demonstruje práci s lokální databází a zachování stavu aplikace.

## Funkcionalita
**Identita hráče**: Vytvoření profilu a přihlášení.

**Herní mechanika**: Kvíz s počítáním skóre a životů (Firewall Integrity).

**Persistence dat**: Aplikace si pamatuje, u které otázky hráč skončil, a umožňuje pokračovat ve hře i po restartu.

**High Score**: Ukládání nejvyššího dosaženého skóre pro každého uživatele.

Poznámka: Aplikace využívá fallbackToDestructiveMigration, při změně verze databáze se data resetují.
